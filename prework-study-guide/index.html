Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git ckeckout "branch name": switches you to that branch.
  • git add -A: used to stage your changes "add -A" commits all files in the working branch.
  • git commit -m "added code to HTML file": commits them to our repository, We use the -m flag to associate a message with our commit.
  • git pull origin main: we need to sync our local branch with the remote branch in GitHub, use this to pull in the base branch.
  • git push origin "branch name": used to push up all of our changes.
